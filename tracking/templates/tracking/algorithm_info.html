Shortest Path Engine

Dijkstra's Algorithm Implementation

The system utilizes a custom implementation of **Dijkstra's algorithm** to determine the most efficient travel path across the transit network.

while pq:
  current_dist, current_stop = heapq.heappop(pq)
  ...
  # Update neighbor distances
  if distance < distances[neighbor]:
    distances[neighbor] = distance
    heapq.heappush(pq, (distance, neighbor, path))
Complexity O(E log V)
Optimization Min-Priority Queue

Geospatial Math

Haversine Distance Model

Used to calculate true physical distance (Great-circle distance) between GPS coordinates over the Earth's curved surface.

a = sin²(Δφ/2) + cos φ₁ ⋅ cos φ₂ ⋅ sin²(Δλ/2)
  • Accounts for Earth's curvature
  • High precision for short distances
  • Used for both path-finding & ETA
